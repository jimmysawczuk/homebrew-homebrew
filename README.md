# Homebrew formulas

## Getting started

```sh
$ brew tap jimmysawczuk/homebrew
$ brew install <formula>
```

## Available formulas

| Project        | Github Repo                                        | `brew` command            |
| -------------- | -------------------------------------------------- | ------------------------- |
| **scm-status** | [Repo](https://github.com/jimmysawczuk/scm-status) | `brew install scm-status` |
